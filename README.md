# chitchat

Building a bullgit chat from scratch

# Install

- install ruby 2.1.2 or newer
- `bundle install`
- `npm install`
- `gulp build`

# Starting the server

`GITHUB_KEY=<the_github_key> GITHUB_SECRET=<the_github_secret> bundle exec rackup`
